# Automatic image Labelizer / Description

A Streamlit application for image labeling and annotation.

## Features

- Upload and label images
- Export labeled data
- Clean and intuitive interface

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the application:

```bash
streamlit run app.py
```
